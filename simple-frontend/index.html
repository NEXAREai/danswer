🛠️ Quick Setup Guide

  1. Use Simple Frontend:
    cd simple-frontend && python3 -m http.server 12000
  2. Or Start Full Development Server:
    ./start-dev-server.sh
  3. For Production Setup:
    ./scripts/setup_and_deploy.sh setup
    ./scripts/setup_and_deploy.sh deploy prod
